# Zombie Horder
Um jogo de horda feito com C++ em 3 dias  

# Mecanicas

## Movimentacao

A movimentação é bem simples **WASD** para andar e para Correr Basta segurar **SHIFT Esquerdo**  
Para atirar é **IJKL** com a mesma direcoes que o **WASD**

## Dificuldade  
***
### O jogo ira peguntar a dificuldade que voce deseja Escolher
- Facil
  - Tempo Necessario Pra sobreviver 30 segundos
  - Cooldown de spawn de inimigos 5 segundos
- Medio
  - Tempo Necessario Pra sobreviver 1 minuto
  - Cooldown de spawn de inimigos 3 segundos
- Dificil
  - Tempo Necessario Pra sobreviver 2 minutos
  - Cooldown de spawn de inimigos 2 segundos
